# Descrizione status.php

## Scopo dello script:

Lo script deve restiruire una stringa in cui viene specificato lo stato in cui si trova il servizio richiesto


## Prerequisiti:

PHP


## Comandi di esecuzione con esempi:

1. php status.php
2. php status.php api
3. php status.php database


## Output atteso:

1. Servizio demo: pronto
2. Servizio api: pronto
3. Servizio database: pronto
